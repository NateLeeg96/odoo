Cloud phone connectors

Work

What does this module solve?

Default behavior

By default, there are no cloud telephony connectors in odoo and employees need to use the personal accounts of telephony providers.

Firstly, they are not always convenient

secondly, they do not have a bunch of employees with numbers

thirdly, as a rule, poor filters and call history

fourth, the storage time of records is limited. And fifth they don't know anything about sales orders

Behavior after install

After installing this module, the basic functionality of connecting to cloud telephony providers is available.

Ready-made models of calls, telephone numbers and connectors. You just need to write and implement the api

of your supplier, everything else is in this module. Namely, each partner can see his incoming and outgoing

calls with the ability to listen online, the same in sale.order and hr.employee moreover,

2 roles are available manager and employee manager can see all calls , filter and search for reports,

manage numbers and add multiple connectors. For example, your campaign may have several telephony providers

that the module supports. Watch the video on the example of one built-in connector

Features

  • Watch sale.order, hr.employee, res.partners real calls and recordings
  • Watch all calls and calls recordings in odoo
  • The ability to add your own connector to the finished modules and make a pull/merge request
  • MTS connector available out of box
